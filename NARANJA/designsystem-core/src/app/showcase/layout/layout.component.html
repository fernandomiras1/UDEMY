Hide element

.hide-in-xs

Oculta un elemento en un breakpoint específico: .hide-in-(breakpoint) (Cambiar el tamaño de la ventana para ver el resultado)

.hide-up-md

Oculta un elemento a partir de un breakpoint en forma ascendente: .hide-up-(breakpoint) (Cambiar el tamaño de la ventana para ver el resultado)

.hide-down-md

Oculta un elemento a partir de un breakpoint en forma descendente: .hide-down-(breakpoint) (Cambiar el tamaño de la ventana para ver el resultado)

Breakpoints

Los breakpoints son rangos de anchos predeterminados que son útiles para generar definiciones particulares de acuerdo al ancho de cada pantalla. Cada breakpoint determina el ancho total de la grilla, como así también el de las calles y columnas.
